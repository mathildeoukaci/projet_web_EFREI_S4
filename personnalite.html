Les pieds Celtics

Si vous avez les pieds Celtics, sachez que votre entourage vous trouve très vulgaire donc il serait temps d’arrêter de faire des doigts d’honneur à tout le monde avec vos pieds.
En plus, vous avez du mal à trouver des chaussures.

Les pieds Germains

Si vous avez les pieds Germains, vous manquez certainement d’ambition et vous préférez fuir les responsabilités. Aller de l’avant est contraire à vos principes, en témoignent vos orteils qui font demi-tour.

Les pieds Grecs

Si vous avez les pieds Grecs, vous ne manquez pas d’ambition mais de concrétisation. Vous adorez tout remettre à demain lorsqu’il s’agit d’atteindre la perfection. La preuve : voilà longtemps que vous auriez déjà dû couper cet orteil qui dépasse !!!

Les pieds Romains

Si vous avez les pieds Romains, vous remettez toujours tout en cause. Vous avez du mal à faire confiance en ce que vous disent les autres. Vous n’aimez pas rentrer dans les codes. La preuve vous remettez en cause la science : la nature vous a donné 5 orteils par pieds, pourquoi vous en utiliser que 3 ?

Les pieds Egyptiens

Si vous avez les pieds Egyptiens, c’est que vous êtes bien ordonné. Les inégalités ne vous posent pas de problème. Mais faites attention : le "petit doigt de pied" pourrait s’allier avec le "doigt de pied moyen" pour faire une révolution contre le "gros doigt de pied". Ce qui vous causerait, par la suite, des problèmes majeurs pour marcher.
